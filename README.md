# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar

```
npm install
```

3.Tener instalado el CLI de Nest

```
npm i -g @nestjs/cli
```

4. Levantar la base de datos

```
docker compose -f docker-compose.yaml up
```

## Stack usado

- MongoDB
- Nest
